Create: Beyond Limits is an addon for the Create mod that expands automation through new recipes, while also introducing independent systems such as Aroma blocks and Sub-Totems.

This project is currently in an early **test build** stage.

---

## Overview

Create: Beyond Limits focuses on:
- Expanding Create-compatible recipes
- Providing new crafting components
- Introducing independent blocks and totems that do not modify Create itself

Recipes are designed to integrate smoothly with Create’s automation systems, while blocks and totems can be used on their own.

---

## Version Information

- **Current Version:** 0.1.0 (Test Build)
- **Minecraft:** 1.20.1
- **Mod Loader:** Forge
- **Required Dependency:** Create **6.0.1 – 6.0.8**

⚠ This mod will **not work** with Create versions outside this range.

---

## Features

- New recipes designed for Create automation
- Aroma block system with tiered effects
- Sub-Totems that grant:
  - Speed
  - Regeneration
  - Strength
- New items used as crafting components and progression materials
- & more "No spoilers"

---

## Content Overview

### Items
- Nether Star Fragment
- Drained Nether Star Fragment
- Unstable Quartz
- Pressurized Core
- Fabricated Elytra Wing
- Wing
- Incomplete Forged Netherite
- Incomplete Copper Ingot
- SOS Signal (lore item)
- Sub Totems (Speed, Regeneration, Strength)

### Blocks
- Gustweave Aroma (Speed – Tier 1)
- Galeheart Aroma (Speed – Tier 2)
- Bloomrest Aroma (Regeneration – Tier 1)
- Vernal Lifeburst Aroma (Regeneration – Tier 2)
- Ironpulse Aroma (Strength – Tier 1)
- Titanblood Aroma (Strength – Tier 2)

---

## Installation

1. Install **Minecraft Forge 1.20.1**
2. Install **Create 6.0.1 – 6.0.8**
3. Place `CreateBeyondLimits.jar` into your `mods/` folder
4. Launch Minecraft

---

## Development Status

This mod is under active development.

- Current builds are **test builds**
- Content, balance, and recipes may change
- Not recommended for permanent worlds without backups

---

## Compatibility Notes

- Blocks and Sub-Totems are **not part of Create**
- Create compatibility applies **only to recipes**
- Recipes are fully compatible with Create automation systems
