## <img src="https://i.imgur.com/8X17XkO.png" width="35"/> Key Features

* **Advanced Beacons:** Evolve Aroma blocks into animated beacons with localized "Blessed" radii.
* **Fragrance & Soul Mechanics:** Utilize Bastion, Cinder, or Malice blocks to bind player souls for unique dimension-wide benefits.
* **Natural Automation:** Enable autonomous animal breeding through internal gamerules, removing the need for manual feeding.
* **New Flora:** Cultivate unique crops like **Rosemary**, integrated into high-tier agricultural processing.
* **Independent Systems:** High-fidelity blocks that expand gameplay depth without modifying Create’s core internal mechanics.
