# Version 0.5.5: The Ascension

### Dependencies
* **GeckoLib 4.4.7** (Now Required)
* **Create**
* **Farmer's Delight**

---

### New Content

#### The Supercharged Sanctum
A new tier of Aroma technology. When a standard Aroma block is struck by lightning, it transforms into a Supercharged Sanctum.
* **Increased Range:** Effective radius expanded to 64 blocks.
* **The Blessed Status:** A powerful new composite effect that grants six buffs simultaneously, including high-tier Speed and Resistance.
* **Redesigned Visuals:** Fully animated models featuring custom golden electric particles and tracking animations.

#### Rosemary Crops
A hardy new herb added to world generation.
* **Natural Spawning:** Found growing in Old Growth Taiga biomes and across Mountain Peaks.
* **Cultivation:** Rosemary can be farmed on both standard Farmland and Podzol.
* **Growth Stages:** Features five distinct visual growth stages from seed to harvest.

---

### Improvements & Changes

#### Model Overhaul
All Aroma-related blocks have received a complete visual redo. Static models have been replaced with high-fidelity animated versions to provide a more immersive experience.

#### Performance Optimization
The mod's internal logic has been overhauled. These structural changes ensure better stability, smoother frame rates during particle-heavy events, and improved compatibility with large modpacks.

#### Advancement System
Added a new challenge: **"Blessed Ascent."** This achievement is earned by successfully supercharging an Aroma block. It remains hidden in the advancement menu until triggered.

---

### Bug Fixes
* Resolved an issue where block particles would occasionally display a placeholder "black and purple" texture.
* Fixed a conflict that caused the game window to lose focus or minimize during specific background processes.
* Corrected loot table errors to ensure Supercharged blocks drop the correct items when harvested with appropriate tools.
* Patched a registry error that caused Aroma blocks to stop functioning after traveling between dimensions.
